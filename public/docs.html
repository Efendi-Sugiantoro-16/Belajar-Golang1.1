Warehouse Management App Documentation

Overview

This Warehouse Management App allows users to manage warehouse inventory efficiently. Users can view, add, edit, and delete inventory items, as well as visualize inventory data through charts.

Features

Inventory Management

View all items in the warehouse, including their name, description, quantity, and location. Items can be edited or deleted directly from the inventory page.

Add New Item

Use the Add Item page to add new inventory items by providing the name, description, quantity, and location.

Edit Item

Edit existing items by clicking the Edit button on the inventory page, which redirects to the Edit Item page.

Charts

Visualize inventory data with bar, line, and pie charts. Select the chart type from the dropdown menu on the Charts page.

API Endpoints

The app communicates with a backend API to manage inventory items. The following endpoints are used:

  • GET /api/items: Retrieve all inventory items.
  • POST /api/items: Add a new inventory item. Requires JSON body with name, description, quantity, and location.
  • PUT /api/items/{id}: Update an existing item by ID. Requires JSON body with updated fields.
  • DELETE /api/items/{id}: Delete an item by ID.

Special Tips & FAQs

How to delete an item?

On the Inventory page, click the Delete button next to the item you want to remove. You will be asked to confirm the deletion.

Can I see real-time updates?

Currently, the app does not support real-time updates. Refresh the Inventory page to see the latest data.

How to change chart types?

On the Charts page, use the dropdown menu to select between Bar, Line, and Pie charts.
